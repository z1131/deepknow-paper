# 环境变量配置清单 (.env)

请在服务器的 `/opt/deepwrite/.env` 文件中配置。

## 1. 通用配置 (ECS A & ECS B 均需)

```properties
# === 阿里云基础服务 ===
MYSQL_HOST=rm-xxxx.mysql.rds.aliyuncs.com
MYSQL_PORT=3306
MYSQL_DB_NAME=deepwrite
MYSQL_USER=root
MYSQL_PASSWORD=你的RDS密码

REDIS_HOST=r-xxxx.redis.rds.aliyuncs.com
REDIS_PORT=6379
REDIS_USERNAME=deepwrite # Redis 登录用户名
REDIS_PASSWORD=你的Redis密码

# === 注册中心 (Nacos) ===
# ECS A 填 172.17.150.78，ECS B 填 172.17.150.78
NACOS_ADDR=172.17.150.78:8848
NACOS_USERNAME=nacos
NACOS_PASSWORD=nacos

# === 应用配置 ===
PROFILE=prod
JWT_SECRET=你的JWT密钥
```

## 2. ECS A 专属配置 (控制节点)

无特殊额外配置。

## 3. ECS B 专属配置 (计算节点)

ECS B 额外需要 AI 相关的 Key：

```properties
# === AI 服务 ===
# 暂时留空，后续补充
GEMINI_API_KEY=
```
