# 数据库设计 - Project

## 表结构: `project`

| 字段名 | 类型 | 长度 | 约束 | 描述 |
| :--- | :--- | :--- | :--- | :--- |
| id | BIGINT | 20 | PK, Auto Inc | 主键 |
| user_id | BIGINT | 20 | Not Null | 归属用户 ID |
| title | VARCHAR | 255 | Nullable | 论文题目 (初始为空) |
| status | VARCHAR | 50 | Not Null | 状态 (INIT, TOPIC_GENERATED, ...) |
| created_at | DATETIME | - | Not Null | 创建时间 |
| updated_at | DATETIME | - | Not Null | 更新时间 |
| deleted | TINYINT | 1 | Default 0 | 逻辑删除 |

## SQL

```sql
CREATE TABLE IF NOT EXISTS `project` (
    `id` BIGINT NOT NULL AUTO_INCREMENT COMMENT '主键ID',
    `user_id` BIGINT NOT NULL COMMENT '用户ID',
    `title` VARCHAR(255) COMMENT '最终确定的论文题目',
    `status` VARCHAR(50) NOT NULL DEFAULT 'INIT' COMMENT '项目状态',
    `created_at` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    `updated_at` DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',
    `deleted` TINYINT(1) NOT NULL DEFAULT 0 COMMENT '逻辑删除',
    PRIMARY KEY (`id`),
    INDEX `idx_user_id` (`user_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='论文写作项目表';
```
