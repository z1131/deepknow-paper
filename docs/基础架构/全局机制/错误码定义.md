# 全局错误码定义

## 错误码格式
```
[业务域]_[错误类型]_[具体代码]
```

## 通用错误码
- `COMMON_001`: 参数验证失败
- `COMMON_002`: 数据库连接失败
- `COMMON_003`: 权限不足
- `COMMON_004`: 资源不存在
- `COMMON_005`: 服务器内部错误

## 业务域错误码
根据具体业务模块定义...

## 错误响应格式
```json
{
  "success": false,
  "error": {
    "code": "COMMON_001",
    "message": "参数验证失败",
    "details": "具体错误信息..."
  }
}
```
